# 부하 테스트

## 대상
### 선착순 쿠폰 발급 기능
> POST /redis/coupon/requestIssue

#### 기능
  - 사용자가 특정(선착순) 쿠폰에 대해 발급 요청을 하면 redis sorted set 에 요청을 저장
  - 발급은 배치 프로세스에 의해 수행
#### 테스트 목적
  - 동시성 환경에서 redis 요청 처리 성능의 측정을 위함.
  - API 안정성 및 실패율을 분석하기 위함.
  - 서버 처리의 한계를 파악하기 위함,

#### 시나리오
- 사용자 수 : 200 명 (가변 vu)
- 테스트 시간 : 30 초
- 목표 :
  - 실패율 < 1%
  - 95% 의 요청에 대한 응답 < 500ms
